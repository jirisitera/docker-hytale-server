# Security Policy

## Supported versions

Only the default branch is guaranteed to receive security updates and fixes.

## Reporting a vulnerability

To privately report vulnerabilities, use GitHub Security Advisories:

- https://github.com/jirisitera/docker-hytale-server/security/advisories/new

Include:

- reproduction steps
- impact assessment (threat level)
- optionally a suggested fix

Vulnerabilities in the **official Hytale server software** itself should be reported to Hypixel Studios via their official channels.

## Secrets & sensitive data

When reporting issues, never include any secrets or tokens.
